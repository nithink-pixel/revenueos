# 💰 RevenueOS — Automated Revenue Leak Detection

🔗 **[Live Demo → nithin-revenueos.streamlit.app](https://nithin-revenueos.streamlit.app)**

> **Find exactly where your business is bleeding money — ranked by dollar impact — with an AI-generated executive action brief.**

Most companies have dashboards. Nobody has a system that *proactively finds* the problems they didn't know to look for and tells leadership what to do about them.

RevenueOS scans your business data across three dimensions, surfaces anomalies that cost real money, quantifies the ARR impact, and uses Claude (Anthropic) to generate a boardroom-ready decision brief in seconds.

## ⚙️ How It Works

### 1. Churn Radar
Scans every customer dimension (segment, plan, channel, region) and computes churn rate per cohort. Any group with churn **1.5x above baseline** is flagged. ARR at risk = churned customers × average MRR × 12.

### 2. Marketing Waste Detector
Calculates blended CAC across all channels. Channels with CAC **≥1.5x blended average** are flagged. Waste = actual spend minus what you'd have paid at blended efficiency.

### 3. Support Bottleneck
Joins ticket response times with customer churn data. Finds the response time threshold where churn spikes. Quantifies ARR at risk from customers who received slow support and churned.

### 4. AI Synthesis (Claude API)
All findings are structured into a JSON payload and passed to Claude with a prompt instructing it to act as a senior strategy consultant. Output is a structured brief: situation → root cause → ranked actions → priority this week.

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/nithink-pixel/revenueos.git
cd revenueos

# 2. Create virtual environment
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Set your Anthropic API key
export ANTHROPIC_API_KEY=your_key_here

# 5. Generate sample data
python sample_data/generate_data.py

# 6. Launch the dashboard
streamlit run app.py
```

Open `http://localhost:8501` in your browser.
